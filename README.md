# expense-tracker
Simple expense tracker as a final project of Web Development II class. This web app will include technologies like React.js, Node.js, PHP and SQL (basically a full-stack project).

As of now I plan to include these screens:
* Home: simple welcome message and buttons to log a new expense or view current expenses.
* New expense: a simple form to log name, amoung, category and date.
* View expenses: visualization of expenses and summary.

With this project I want to start learning React and Node, which are essential technologies I need to learn as a front-end developer.
